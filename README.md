# Mr Lato — Portfolio Vidéaste & Créateur de Contenu

Portfolio personnel de **Adéyemi Abdou Latif OLOUDE**, vidéaste mobile et créateur de contenu basé à **Cotonou, Bénin**.

## Présentation

Site vitrine one-page présentant les réalisations, services et coordonnées de Mr Lato. Conçu pour convertir les visiteurs en clients — avec un copywriting orienté bénéfices et des références clients nommées (MTN BÉNIN, Guinness World Records, Choiseul Africa Summit, Benin Investment Forum).

---

## Structure du projet

```
Projet3-Portfolio/
├── index.html              # Page principale (one-page)
├── style.css               # Feuille de styles
├── script.js               # Logique JS (filtres, lightbox, animations)
├── favicon.ico             # Icône du site
├── img/                    # Photos du portfolio (JPEG haute résolution)
├── optimized-thumbs/       # Miniatures optimisées (WebP) pour le chargement rapide
└── video/                  # Vidéos locales (MP4)
```

---

## Stack technique

- **HTML / CSS / JavaScript** — pur, sans framework
- **Font Awesome 6.5** — icônes
- **Google Fonts** — Space Grotesk
- **Lightbox custom** — lecture vidéos, photos et TikTok embarqués
- **Filtre de galerie** — JS natif avec animation CSS
- **IntersectionObserver** — animations au défilement
- **Formspree** — gestion du formulaire de contact
- **Lazy loading** — chargement différé des images (`loading="lazy"`)
- **WebP** — miniatures optimisées pour la performance
